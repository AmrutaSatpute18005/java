This project is a Java-based Hotel Management System developed using Java Swing for the graphical user interface. It allows users to book hotel rooms by entering guest details, selecting room packages, choosing meal plans, and generating a booking receipt.
The application includes real-time input validation, dynamic price calculation, and an automatically generated receipt window after successful booking. It is designed as a desktop application and demonstrates the use of Java GUI components, event handling, and basic business logic.
